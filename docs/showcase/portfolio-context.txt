Short portfolio description:
StockPilot is a full-stack inventory management web application for small business operations. It helps teams manage products, categories, suppliers, stock levels, stock movement history, sales orders, users, roles, low-stock alerts, and inventory dashboard metrics.

Long portfolio description:
Built a full-stack inventory management system using ASP.NET Core MVC, Entity Framework Core, PostgreSQL, ASP.NET Core Identity, Razor Views, Bootstrap, and custom CSS. The application includes role-based authorization for Admin, Manager, and Staff users, CRUD workflows for inventory records, product search and filtering, dashboard statistics, low-stock and out-of-stock alerts, stock movement tracking, order workflows, audit logs, seeded demo data, Docker support, and deployment on Render with PostgreSQL.
